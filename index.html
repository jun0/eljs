eelll/JS とは

T-Code 練習プログラム EELLL (うなぎぎぎ) の JavaScript 版です。 TUT-Code にも対応しています。 MSIE6 や Netscape7.1 などで動作するものと思われます。 要 JavaScript。Cookie 推奨。

  • → オンライン版 eelll/JS で練習してみる。 ナローバンドな人は、読みこみにかなり時間がかかる予感がします。 覚悟のほどを。 G-Code/Try-Code/TT-Code な人は、 full version をどうぞ。

  • → お持ち帰り版 eljs01.zip をダウンロード。オフラインで練習したい人はこちらをどうぞ。 230KB くらい。

使い方

  1. eelll/JS のページに進む。 ページの読みこみが完了したら、オフラインにしてかまいません。 IME や 漢直Win は OFF にしておきます。

  2. TUT-Code な人は、[設定] セクションの [入力方式] 欄で [TUT-Code] を、 [練習テキスト] 欄で [kanji_a1.txt] を、 それぞれ選択し、[設定] ボタンを押す。 T-Code な人は、そのままで OK です。

  3. [メイン] セクションの入力窓 (細長いテキストフィールド) をクリックしてフォーカスを移す。 心の準備ができたら、リターンキーを押してください。 レッスンが始まります。

  4. 練習テキストが一行ずつ表示されますので、 T-Code または TUT-Code で入力してください。 入力したキーは、画面には表示されません。 まちがいを気にせずに (BackSpace で消したりせずに) どんどん入力しましょう。

  5. 一行分の入力が終わったら、リターンキーを押してください。 入力したキーがチェックされ、結果が表示されます。 水色は正しく入力した文字、 赤色はまちがった文字です。

  6. レッスンの最後まで来ると、『もう一度トライしますか?』と 訊かれますので、 同じレッスンをもう一度練習したいなら r と入力してリターンキーを、 次のレッスンに進みたいなら n と入力してリターンキーを押します。 また、q と入力してリターンキーを押すと、 練習を終了し、総合成績を表示します。
